# Audio Files for Smart EduBuddy

## Required Audio Files

You need to add two audio files to this directory:

1. **correct.mp3** - Played when answer is correct
2. **wrong.mp3** - Played when answer is wrong

---

## Option 1: Download Free Sound Effects

### Websites for Free Sounds:

**Freesound.org**
- https://freesound.org/
- Search: "correct", "success", "bell", "ding"
- Search: "wrong", "error", "buzzer"
- Download as MP3

**Zapsplat**
- https://www.zapsplat.com/
- Search in "Game Sounds" category
- Positive sounds for correct
- Negative sounds for wrong

**Mixkit**
- https://mixkit.co/free-sound-effects/
- Filter by "Game" or "Notification"
- High-quality, royalty-free

---

## Option 2: Record Your Own Voice

### Using Audacity (Free Software):

**1. Download Audacity:**
- https://www.audacityteam.org/

**2. Record:**
- Click red record button
- Say "Correct! Great job!" clearly
- Stop recording

**3. Edit:**
- Remove silence at start/end
- Normalize volume
- Export as MP3

**4. Save:**
- Save as `correct.mp3` in this folder
- Repeat for wrong.mp3

### Suggested Phrases:

**For correct.mp3:**
- "Correct! Great job!"
- "Well done!"
- "You got it!"
- "That's right!"
- "Excellent!"

**For wrong.mp3:**
- "Oops! Try again!"
- "Not quite, try once more!"
- "Almost! Give it another try!"
- "Good try! Want to try again?"

## Option 3: Text-to-Speech (Quick & Easy)

### Using Google Text-to-Speech:

## Technical Requirements

**Format:** MP3 (recommended)
- Also supports: WAV, OGG

**Quality:**
- Sample Rate: 44100 Hz (standard)
- Bit Rate: 128 kbps or higher
- Duration: 1-3 seconds recommended
- File Size: Keep under 500 KB each

**Testing:**
- Play file on your computer first
- Ensure clear and loud enough
- No background noise
- Appropriate for children

---

## File Naming (Important!)

Files MUST be named exactly:
- `correct.mp3` (all lowercase)
- `wrong.mp3` (all lowercase)

Do NOT use:
- Correct.mp3
- CORRECT.mp3
- correct_sound.mp3
- any other name

---

## Installation

1. Download or create your MP3 files
2. Rename them to `correct.mp3` and `wrong.mp3`
3. Copy both files to this folder:
   `smart-edubuddy/assets/audio/`
4. Refresh your web dashboard
5. Test by sending a question and answering

---

## Troubleshooting

**Sound doesn't play:**
1. Check file names are exactly correct
2. Check files are in correct folder
3. Click on webpage first (browsers block autoplay)
4. Check speaker volume
5. Try different browser (Chrome recommended)

**Sound is too quiet/loud:**
1. Adjust volume in browser/system
2. Or edit MP3 with audio software
3. Normalize audio to -3dB

**Sound cuts off:**
1. Add 0.2s silence at end
2. Export with different settings
3. Check file isn't corrupted
